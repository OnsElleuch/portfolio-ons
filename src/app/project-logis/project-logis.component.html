Logis Description Generator

Web Development, NLP, 06/2021-08/2021

This Project is a web application for real estate description automatic generation.


The project was during my summer internship 2021 in Logis. It's an R&D project where I got acquinted to the state-of-the-art models for text generation. I worked with OpenAi GPT-3, Google-T5, GPT-Neo ...

In this project I had the chance to:

    Scrape data from Dubaian, Tunisian and French real estate websites using Python.
    Process data and feature engineering.
    Implement a template-based bot that generates real estate property descriptions using Laravel and Angular.
    Train a model to generate descriptions based on keywords.
    First experience with AWS sagemaker.
    Deploy the module on Heroku.
